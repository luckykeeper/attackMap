# 简介

[前往导航](#导航)

你可能看过很多酷炫的网络攻击地图，就像卡巴斯基的[这个](https://cybermap.kaspersky.com/)，但是一直没有什么开源的、能够自建的网络攻击地图，本项目就能为你带来为你的网络量身定做的网络攻击地图
